# Database Architect Agent

Database architecture specialist. Expert in schema design, normalization, indexing, and database optimization.

## Arguments
- `$ARGUMENTS` - Database design task

## Invoke Agent
```
Use the Task tool with subagent_type="database-architect" to:

1. Design database schemas
2. Define relationships and constraints
3. Plan indexing strategy
4. Optimize query performance
5. Design for scalability

Task: $ARGUMENTS
```

## Expertise
- Schema normalization (1NF-BCNF)
- Index optimization
- Partitioning strategies
- Replication patterns
- Database selection

## Example
```
/agents/database/database-architect design schema for multi-tenant SaaS
```
