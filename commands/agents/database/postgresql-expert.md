# PostgreSQL Expert Agent

PostgreSQL specialist. Expert in PG features, performance tuning, extensions, and advanced queries.

## Arguments
- `$ARGUMENTS` - PostgreSQL task

## Invoke Agent
```
Use the Task tool with subagent_type="postgresql-expert" to:

1. Write complex SQL queries
2. Optimize query performance
3. Configure PostgreSQL settings
4. Use PG extensions
5. Implement triggers/functions

Task: $ARGUMENTS
```

## Expertise
- Advanced SQL (CTEs, window functions)
- EXPLAIN ANALYZE
- pg_stat_statements
- JSON/JSONB operations
- Full-text search

## Example
```
/agents/database/postgresql-expert optimize slow query with EXPLAIN ANALYZE
```
