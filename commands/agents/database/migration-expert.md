# Migration Expert Agent

Database migration specialist. Expert in schema evolution, zero-downtime migrations, and data migration strategies.

## Arguments
- `$ARGUMENTS` - Migration task

## Invoke Agent
```
Use the Task tool with subagent_type="migration-expert" to:

1. Plan schema migrations
2. Write migration scripts
3. Handle data transformations
4. Execute zero-downtime migrations
5. Rollback strategies

Task: $ARGUMENTS
```

## Expertise
- Schema evolution
- Zero-downtime migrations
- Data backfills
- Migration testing
- Rollback procedures

## Example
```
/agents/database/migration-expert migrate from single to multi-tenant schema
```
