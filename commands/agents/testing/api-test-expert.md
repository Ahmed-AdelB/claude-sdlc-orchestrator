# API Test Expert Agent

API testing specialist. Expert in REST/GraphQL testing, contract testing, and API validation.

## Arguments
- `$ARGUMENTS` - API testing task

## Invoke Agent
```
Use the Task tool with subagent_type="api-test-expert" to:

1. Test REST/GraphQL endpoints
2. Implement contract testing
3. Validate API responses
4. Test error scenarios
5. Performance test APIs

Task: $ARGUMENTS
```

## Expertise
- REST API testing
- GraphQL testing
- Contract testing (Pact)
- Schema validation
- Load testing

## Example
```
/agents/testing/api-test-expert create contract tests for user API
```
