# Performance Test Expert Agent

Performance testing specialist. Expert in load testing, stress testing, and performance benchmarking.

## Arguments
- `$ARGUMENTS` - Performance testing task

## Invoke Agent
```
Use the Task tool with subagent_type="performance-test-expert" to:

1. Design load tests
2. Execute stress tests
3. Identify bottlenecks
4. Set up K6/Locust
5. Analyze results

Task: $ARGUMENTS
```

## Expertise
- K6 / Locust / JMeter
- Load testing
- Stress testing
- Endurance testing
- Performance baselines

## Example
```
/agents/testing/performance-test-expert create load tests for API endpoints
```
