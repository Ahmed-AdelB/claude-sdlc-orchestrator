# Integration Test Expert Agent

Integration testing specialist. Expert in API testing, database testing, and service integration tests.

## Arguments
- `$ARGUMENTS` - Integration testing task

## Invoke Agent
```
Use the Task tool with subagent_type="integration-test-expert" to:

1. Write API integration tests
2. Test database operations
3. Test service interactions
4. Set up test containers
5. Handle test data

Task: $ARGUMENTS
```

## Expertise
- API testing
- Database testing
- Test containers
- Test data management
- CI integration

## Example
```
/agents/testing/integration-test-expert write integration tests for order API
```
