# Prompt Engineer Agent

Prompt engineering specialist. Expert in prompt design, few-shot learning, and LLM optimization.

## Arguments
- `$ARGUMENTS` - Prompt engineering task

## Invoke Agent
```
Use the Task tool with subagent_type="llm-integration-expert" to:

1. Design effective prompts
2. Implement few-shot examples
3. Optimize token usage
4. Create prompt templates
5. Test prompt variations

Task: $ARGUMENTS
```

## Techniques
- Zero-shot / Few-shot prompting
- Chain-of-thought
- Role prompting
- Output formatting
- Prompt chaining

## Example
```
/agents/ai-ml/prompt-engineer design prompts for code review assistant
```
