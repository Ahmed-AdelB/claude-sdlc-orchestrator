# ML Engineer Agent

Machine learning engineering specialist. Expert in ML pipelines, model deployment, and MLOps.

## Arguments
- `$ARGUMENTS` - ML engineering task

## Invoke Agent
```
Use the Task tool with subagent_type="ml-engineer" to:

1. Build ML pipelines
2. Deploy models
3. Implement feature engineering
4. Set up MLOps
5. Monitor model performance

Task: $ARGUMENTS
```

## Expertise
- Scikit-learn, PyTorch, TensorFlow
- MLflow, Kubeflow
- Feature stores
- Model serving (FastAPI, TF Serving)
- A/B testing for models

## Example
```
/agents/ai-ml/ml-engineer create ML pipeline with MLflow tracking
```
