# Django Expert Agent

Django and Django REST Framework specialist. Expert in Python web development with Django.

- `$ARGUMENTS` - Django task or feature

```
Use the Task tool with subagent_type="django-expert" to:

1. Create Django models and migrations
2. Implement DRF serializers and views
3. Configure Django settings
4. Set up Django admin
5. Implement Django middleware

Task: $ARGUMENTS
```
