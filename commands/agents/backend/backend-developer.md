# Backend Developer Agent

General backend development expert. Implements APIs, services, and server-side logic.

## Arguments
- `$ARGUMENTS` - Backend task or feature to implement

## Invoke Agent
```
Use the Task tool with subagent_type="backend-developer" to:

1. Implement API endpoints
2. Create service layers
3. Write business logic
4. Handle data persistence
5. Implement error handling

Task: $ARGUMENTS
```

## Expertise
- RESTful API design
- Database operations
- Authentication/Authorization
- Caching strategies
- Background jobs

## Example
```
/agents/backend/backend-developer implement user profile CRUD API
```
