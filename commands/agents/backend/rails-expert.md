# Rails Expert Agent

Ruby on Rails specialist. Expert in Rails conventions, ActiveRecord, and Ruby best practices.

## Arguments
- `$ARGUMENTS` - Rails task or feature

## Invoke Agent
```
Use the Task tool with subagent_type="rails-expert" to:

1. Create Rails models and migrations
2. Implement controllers and routes
3. Set up ActiveRecord associations
4. Configure Rails middleware
5. Implement background jobs (Sidekiq)

Task: $ARGUMENTS
```

## Expertise
- Rails conventions
- ActiveRecord patterns
- Rails API mode
- Action Cable (WebSockets)
- RSpec testing

## Example
```
/agents/backend/rails-expert create REST API with Rails 7 API mode
```
