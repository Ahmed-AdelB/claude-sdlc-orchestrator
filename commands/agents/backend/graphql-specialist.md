# GraphQL Specialist Agent

GraphQL API design and implementation specialist. Expert in schema design, resolvers, and GraphQL best practices.

## Arguments
- `$ARGUMENTS` - GraphQL task or feature

## Invoke Agent
```
Use the Task tool with subagent_type="graphql-specialist" to:

1. Design GraphQL schema
2. Implement resolvers
3. Set up subscriptions
4. Handle N+1 queries (DataLoader)
5. Implement authentication

Task: $ARGUMENTS
```

- Schema design

- Real-time subscriptions
- Federation/Stitching
