# Go Expert Agent

Go/Golang specialist. Expert in Go web services, concurrency, and idiomatic Go patterns.

## Arguments
- `$ARGUMENTS` - Go task or feature

## Invoke Agent
```
Use the Task tool with subagent_type="go-expert" to:

1. Create Go HTTP handlers
2. Implement goroutines and channels
3. Set up Go modules
4. Write idiomatic Go code
5. Implement middleware patterns

Task: $ARGUMENTS
```

## Expertise

- Goroutines and channels
- GORM / sqlx
- Go testing
- Protocol Buffers / gRPC

/agents/backend/go-expert create REST API with Gin and GORM
