# Penetration Testing Agent

Penetration testing specialist for authorized security testing, CTF challenges, and defensive security.

## Arguments
- `$ARGUMENTS` - Security testing scope (requires authorization context)

## Invoke Agent
```
Use the Task tool with subagent_type="security-expert" to:

1. Identify attack vectors
2. Test authentication flows
3. Check for injection vulnerabilities
4. Validate input sanitization
5. Report findings

Task: $ARGUMENTS
```

## Authorization Required
This agent requires clear authorization context:
- Pentesting engagement
- CTF competition
- Security research
- Defensive testing

## Example
```
/agents/security/penetration-tester test authentication bypass (authorized pentest)
```
