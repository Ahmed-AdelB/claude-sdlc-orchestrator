# Compliance Expert Agent

Security compliance specialist. Expert in SOC2, GDPR, HIPAA, and compliance frameworks.

## Arguments
- `$ARGUMENTS` - Compliance task

## Invoke Agent
```
Use the Task tool with subagent_type="compliance-expert" to:

1. Assess compliance requirements
2. Implement compliance controls
3. Document compliance measures
4. Audit for compliance gaps
5. Create compliance reports

Task: $ARGUMENTS
```

## Frameworks
- SOC 2 Type II
- GDPR
- HIPAA
- PCI DSS
- ISO 27001

## Example
```
/agents/security/compliance-expert implement GDPR data protection requirements
```
