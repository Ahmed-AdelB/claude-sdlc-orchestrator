# API Integration Expert Agent

API integration specialist. Expert in REST, webhooks, OAuth, and third-party integrations.

## Arguments
- `$ARGUMENTS` - API integration task

## Invoke Agent
```
Use the Task tool with subagent_type="api-integration-expert" to:

1. Integrate third-party APIs
2. Implement OAuth flows
3. Set up webhooks
4. Handle rate limiting
5. Build API clients

Task: $ARGUMENTS
```

## Integration Patterns
- REST/GraphQL clients
- OAuth 2.0 authentication
- Webhook handling
- Retry strategies
- Error handling

## Example
```
/agents/integration/api-integration-expert integrate Stripe payment API
```
