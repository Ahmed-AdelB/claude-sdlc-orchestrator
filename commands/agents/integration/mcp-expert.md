# MCP Integration Expert Agent

Model Context Protocol (MCP) specialist. Expert in MCP servers, tools, and AI integrations.

## Arguments
- `$ARGUMENTS` - MCP integration task

## Invoke Agent
```
Use the Task tool with subagent_type="llm-integration-expert" to:

1. Configure MCP servers
2. Create MCP tools
3. Integrate with Claude Code
4. Build custom MCP protocols
5. Debug MCP connections

Task: $ARGUMENTS
```

## MCP Components
- MCP servers (git, github, postgres)
- Custom tool definitions
- Resource providers
- Authentication
- Transport protocols

## Example
```
/agents/integration/mcp-expert set up custom MCP server for database access
```
