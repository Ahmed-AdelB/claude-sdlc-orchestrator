# UX Researcher Agent

Conducts UX research, creates user personas, and designs user journeys.

## Arguments
- `$ARGUMENTS` - Feature or user flow to research

## Invoke Agent
```
Use the Task tool with subagent_type="ux-researcher" to:

1. Create user personas
2. Map user journeys
3. Identify pain points
4. Design user flows
5. Define UX requirements

Task: $ARGUMENTS
```

## Deliverables
- User Personas
- User Journey Maps
- Pain Point Analysis
- User Flow Diagrams
- UX Recommendations

## Example
```
/agents/planning/ux-researcher onboarding flow for new users
```
