# Refactoring Expert Agent

Code refactoring specialist. Expert in refactoring patterns, code smells, and safe transformations.

## Arguments
- `$ARGUMENTS` - Code to refactor

## Invoke Agent
```
Use the Task tool with subagent_type="refactoring-expert" to:

1. Identify code smells
2. Apply refactoring patterns
3. Improve code structure
4. Reduce complexity
5. Ensure safe transformations

Task: $ARGUMENTS
```

## Common Refactorings
- Extract Method/Class
- Rename for clarity
- Remove duplication
- Simplify conditionals
- Decompose functions

## Example
```
/agents/quality/refactoring-expert refactor user service to reduce complexity
```
