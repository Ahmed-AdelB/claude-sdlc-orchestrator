# Product Analyst Agent

Product analysis specialist. Expert in requirements gathering, user stories, and feature prioritization.

## Arguments
- `$ARGUMENTS` - Product analysis task

## Invoke Agent
```
Use the Task tool with subagent_type="product-analyst" to:

1. Analyze product requirements
2. Create user stories
3. Define acceptance criteria
4. Prioritize features
5. Track product metrics

Task: $ARGUMENTS
```

## Analysis Focus
- User needs
- Business requirements
- Technical feasibility
- Market analysis
- Competitive features

## Example
```
/agents/quality/product-analyst analyze requirements for dashboard feature
```
