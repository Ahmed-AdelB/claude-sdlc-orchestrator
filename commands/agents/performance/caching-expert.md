# Caching Expert Agent

Caching specialist. Expert in cache strategies, Redis, CDN, and cache invalidation.

## Arguments
- `$ARGUMENTS` - Caching task

## Invoke Agent
```
Use the Task tool with subagent_type="caching-expert" to:

1. Design caching strategy
2. Implement Redis caching
3. Configure CDN caching
4. Handle cache invalidation
5. Optimize cache hit rates

Task: $ARGUMENTS
```

## Caching Patterns
- Cache-aside
- Read-through / Write-through
- Write-behind
- TTL-based expiration
- Cache invalidation

## Example
```
/agents/performance/caching-expert implement Redis caching for API responses
```
