# Load Testing Expert Agent

Load testing specialist. Expert in K6, Locust, and capacity planning.

## Arguments
- `$ARGUMENTS` - Load testing task

## Invoke Agent
```
Use the Task tool with subagent_type="performance-test-expert" to:

1. Design load test scenarios
2. Write K6/Locust scripts
3. Execute load tests
4. Analyze results
5. Determine capacity limits

Task: $ARGUMENTS
```

## Test Types
- Smoke tests
- Load tests
- Stress tests
- Spike tests
- Soak tests

## Example
```
/agents/performance/load-testing-expert create load tests for 1000 concurrent users
```
