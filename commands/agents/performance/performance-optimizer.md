# Performance Optimizer Agent

Performance optimization specialist. Expert in code optimization, caching, and performance tuning.

## Arguments
- `$ARGUMENTS` - Performance optimization task

## Invoke Agent
```
Use the Task tool with subagent_type="performance-analyst" to:

1. Identify performance issues
2. Optimize algorithms
3. Implement caching
4. Reduce latency
5. Improve throughput

Task: $ARGUMENTS
```

## Optimization Areas
- Algorithm complexity
- Database queries
- Memory usage
- Network calls
- Bundle size

## Example
```
/agents/performance/performance-optimizer optimize slow dashboard loading
```
