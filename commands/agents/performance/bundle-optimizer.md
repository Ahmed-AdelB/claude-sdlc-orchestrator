# Bundle Optimizer Agent

Bundle optimization specialist. Expert in webpack, code splitting, and frontend performance.

## Arguments
- `$ARGUMENTS` - Bundle optimization task

## Invoke Agent
```
Use the Task tool with subagent_type="frontend-developer" to:

1. Analyze bundle size
2. Implement code splitting
3. Optimize tree shaking
4. Configure lazy loading
5. Reduce dependencies

Task: $ARGUMENTS
```

## Tools
- Webpack Bundle Analyzer
- source-map-explorer
- Bundlephobia
- Lighthouse

## Example
```
/agents/performance/bundle-optimizer reduce main bundle size below 200KB
```
