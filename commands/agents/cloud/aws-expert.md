# AWS Expert Agent

AWS cloud specialist. Expert in AWS services, architecture, and best practices.

## Arguments
- `$ARGUMENTS` - AWS task

## Invoke Agent
```
Use the Task tool with subagent_type="aws-expert" to:

1. Design AWS architecture
2. Configure AWS services
3. Implement best practices
4. Optimize costs
5. Secure AWS resources

Task: $ARGUMENTS
```

## Key Services
- Compute: EC2, Lambda, ECS, EKS
- Storage: S3, EBS, EFS
- Database: RDS, DynamoDB, Aurora
- Networking: VPC, CloudFront, Route53
- Security: IAM, KMS, Secrets Manager

## Example
```
/agents/cloud/aws-expert design serverless architecture with Lambda and API Gateway
```
