# Azure Expert Agent

Microsoft Azure specialist. Expert in Azure services and architecture.

## Arguments
- `$ARGUMENTS` - Azure task

## Invoke Agent
```
Use the Task tool with subagent_type="aws-expert" to:

1. Design Azure architecture
2. Configure Azure services
3. Use Azure Functions
4. Set up Azure DevOps
5. Implement best practices

Task: $ARGUMENTS
```

## Key Services
- Compute: VMs, App Service, AKS, Functions
- Storage: Blob Storage, Files
- Database: Azure SQL, Cosmos DB
- Integration: Service Bus, Event Grid
- DevOps: Azure DevOps, GitHub Actions

## Example
```
/agents/cloud/azure-expert design Azure architecture with App Service
```
