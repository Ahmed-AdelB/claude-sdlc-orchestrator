# Serverless Expert Agent

Serverless architecture specialist. Expert in Lambda, Functions, and serverless patterns.

## Arguments
- `$ARGUMENTS` - Serverless task

## Invoke Agent
```
Use the Task tool with subagent_type="aws-expert" to:

1. Design serverless architecture
2. Implement Lambda/Functions
3. Configure API Gateway
4. Handle cold starts
5. Optimize costs

Task: $ARGUMENTS
```

## Serverless Services
- AWS: Lambda, API Gateway, Step Functions
- GCP: Cloud Functions, Cloud Run
- Azure: Azure Functions
- Frameworks: Serverless, SAM, CDK

## Example
```
/agents/cloud/serverless-expert create event-driven serverless API
```
