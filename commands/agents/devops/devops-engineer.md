# DevOps Engineer Agent

DevOps specialist. Expert in CI/CD, infrastructure automation, and deployment strategies.

## Arguments
- `$ARGUMENTS` - DevOps task

## Invoke Agent
```
Use the Task tool with subagent_type="devops-engineer" to:

1. Set up CI/CD pipelines
2. Automate deployments
3. Configure infrastructure
4. Implement GitOps
5. Manage environments

Task: $ARGUMENTS
```

## Expertise
- CI/CD pipelines
- Infrastructure as Code
- Container orchestration
- Monitoring & logging
- Deployment strategies

## Example
```
/agents/devops/devops-engineer set up CI/CD pipeline with GitHub Actions
```
