# CI/CD Expert Agent

CI/CD specialist. Expert in pipeline design, build automation, and deployment strategies.

## Arguments
- `$ARGUMENTS` - CI/CD task

## Invoke Agent
```
Use the Task tool with subagent_type="devops-engineer" to:

1. Design CI/CD pipelines
2. Implement build stages
3. Configure deployment strategies
4. Set up quality gates
5. Automate releases

Task: $ARGUMENTS
```

## Deployment Strategies
- Blue-green deployment
- Canary releases
- Rolling updates
- Feature flags
- GitOps

## Example
```
/agents/devops/ci-cd-expert implement blue-green deployment pipeline
```
