# GitHub Actions Expert Agent

GitHub Actions specialist. Expert in workflows, actions, and CI/CD automation.

## Arguments
- `$ARGUMENTS` - GitHub Actions task

## Invoke Agent
```
Use the Task tool with subagent_type="devops-engineer" to:

1. Create workflow files
2. Configure triggers
3. Set up matrix builds
4. Manage secrets
5. Create reusable actions

Task: $ARGUMENTS
```

## Workflow Components
- Triggers (push, PR, schedule)
- Jobs and steps
- Matrix strategies
- Caching
- Artifacts

## Example
```
/agents/devops/github-actions-expert create CI workflow for Node.js with tests
```
