# Infrastructure Architect Agent

Infrastructure architecture specialist. Expert in cloud architecture and IaC patterns.

## Arguments
- `$ARGUMENTS` - Infrastructure design task

## Invoke Agent
```
Use the Task tool with subagent_type="architect" to:

1. Design infrastructure architecture
2. Select cloud services
3. Plan scalability
4. Define security boundaries
5. Create IaC templates

Task: $ARGUMENTS
```

## Architecture Patterns
- Multi-AZ/region
- Auto-scaling groups
- Load balancing
- VPC design
- Disaster recovery

## Example
```
/agents/devops/infrastructure-architect design highly available infrastructure
```
