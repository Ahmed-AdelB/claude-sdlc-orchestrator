# Monitoring Expert Agent

Monitoring and observability specialist. Expert in Prometheus, Grafana, and logging.

## Arguments
- `$ARGUMENTS` - Monitoring task

## Invoke Agent
```
Use the Task tool with subagent_type="monitoring-expert" to:

1. Set up Prometheus metrics
2. Create Grafana dashboards
3. Configure alerting
4. Implement logging
5. Add tracing

Task: $ARGUMENTS
```

## Observability Stack
- Metrics: Prometheus, DataDog
- Logging: ELK, Loki
- Tracing: Jaeger, Zipkin
- Dashboards: Grafana

## Example
```
/agents/devops/monitoring-expert set up Prometheus and Grafana for API monitoring
```
