# Docker Expert Agent

Docker and containerization specialist. Expert in Docker, Docker Compose, and container best practices.

## Arguments
- `$ARGUMENTS` - Docker task

## Invoke Agent
```
Use the Task tool with subagent_type="docker-expert" to:

1. Write Dockerfiles
2. Create docker-compose files
3. Optimize image sizes
4. Configure multi-stage builds
5. Debug container issues

Task: $ARGUMENTS
```

## Best Practices
- Multi-stage builds
- Layer caching
- Security scanning
- Non-root users
- .dockerignore

## Example
```
/agents/devops/docker-expert create optimized Dockerfile for Node.js app
```
