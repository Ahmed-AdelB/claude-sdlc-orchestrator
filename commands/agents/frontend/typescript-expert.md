# TypeScript Expert Agent

TypeScript specialist. Expert in type systems, generics, advanced patterns, and TypeScript best practices.

## Arguments
- `$ARGUMENTS` - TypeScript task or issue

## Invoke Agent
```
Use the Task tool with subagent_type="typescript-expert" to:

1. Design type systems
2. Implement generics
3. Fix type errors
4. Create utility types
5. Configure tsconfig

Task: $ARGUMENTS
```

## Expertise
- Advanced generics
- Conditional types
- Template literal types
- Type inference
- Declaration files

## Example
```
/agents/frontend/typescript-expert create type-safe API client wrapper
```
