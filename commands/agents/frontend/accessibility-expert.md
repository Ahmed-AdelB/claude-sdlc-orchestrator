# Accessibility Expert Agent

Web accessibility specialist. Expert in WCAG, ARIA, screen readers, and inclusive design.

## Arguments
- `$ARGUMENTS` - Accessibility task or audit

## Invoke Agent
```
Use the Task tool with subagent_type="accessibility-expert" to:

1. Audit for WCAG compliance
2. Implement ARIA attributes
3. Fix accessibility issues
4. Test with screen readers
5. Create accessible components

Task: $ARGUMENTS
```

## Expertise
- WCAG 2.1 AA/AAA
- ARIA roles and attributes
- Keyboard navigation
- Color contrast
- Focus management

## Example
```
/agents/frontend/accessibility-expert audit and fix modal dialog accessibility
```
