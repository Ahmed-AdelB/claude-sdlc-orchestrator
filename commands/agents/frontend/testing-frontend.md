# Frontend Testing Agent

Frontend testing specialist. Expert in Jest, Testing Library, Playwright, and component testing.

## Arguments
- `$ARGUMENTS` - Frontend testing task

## Invoke Agent
```
Use the Task tool with subagent_type="testing-frontend" to:

1. Write unit tests (Jest)
2. Create component tests (Testing Library)
3. Implement E2E tests (Playwright)
4. Test custom hooks
5. Set up test coverage

Task: $ARGUMENTS
```

## Expertise
- Jest / Vitest
- React Testing Library
- Playwright / Cypress
- Mock Service Worker (MSW)
- Snapshot testing

## Example
```
/agents/frontend/testing-frontend write tests for authentication flow
```
