# Frontend Developer Agent

General frontend development specialist. Expert in modern web development, responsive design, and UI implementation.

## Arguments
- `$ARGUMENTS` - Frontend task or feature

## Invoke Agent
```
Use the Task tool with subagent_type="frontend-developer" to:

1. Implement UI components
2. Create responsive layouts
3. Handle state management
4. Integrate with APIs
5. Optimize performance

Task: $ARGUMENTS
```

## Expertise
- HTML5/CSS3/JavaScript
- Component architecture
- Responsive design
- Accessibility (a11y)
- Browser compatibility

## Example
```
/agents/frontend/frontend-developer create responsive product card component
```
