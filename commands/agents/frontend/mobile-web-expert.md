# Mobile Web Expert Agent

Mobile web and PWA specialist. Expert in responsive design, PWA, mobile performance, and touch interactions.

## Arguments
- `$ARGUMENTS` - Mobile web task

## Invoke Agent
```
Use the Task tool with subagent_type="mobile-web-expert" to:

1. Create PWA manifests
2. Implement service workers
3. Optimize mobile performance
4. Handle touch interactions
5. Create responsive layouts

Task: $ARGUMENTS
```

## Expertise
- Progressive Web Apps
- Service Workers
- Mobile-first design
- Touch gestures
- Offline functionality

## Example
```
/agents/frontend/mobile-web-expert convert app to PWA with offline support
```
